A machine learning and AI-powered web application that analyzes Amazon customer reviews, detects sentiment, identifies issue types, and generates automated customer support responses.

**Features** <br>
-Sentiment analysis of customer reviews (Positive / Negative / Neutral) <br>
-Issue detection (Delivery, Refund, Product-related issues) <br>
-Priority classification (High, Medium, Low) <br>
-AI-generated customer support responses using Groq LLaMA 3 <br>
-Interactive Streamlit web interface for real-time analysis <br>

**Tech Stack** <br>
-Python <br>
-Scikit-learn <br>
-TF-IDF Vectorization <br>
-Streamlit <br> 
-Groq LLaMA 3 API <br>
-Support Vector Classifier <br>

**Model Performance** <br>
-Achieved approximately 90% accuracy using Support Vector Classifier (SVC)

**Model Comparison** <br>
Multiple machine learning models were evaluated for sentiment classification, including Linear SVM, Naive Bayes, and Random Forest. Based on performance analysis, SVC was selected as it achieved the highest accuracy (~90%) and better generalization on the dataset.
